Stack Styles

There are two ways to create Stack Icons:
  • When you want to add stacked icons inline with the text:

    In those cases use wrap the two icons inside a div with the crf-inline-stack style
    And on each individual font add the crf-inline-stack-1x style
  • When you want to add stacked icons inside blocked elements:
    MIC
    MUTE
    VOLUME LO
    VOLUME HI
    In those cases use wrap the two icons inside a div with the crf-grid-stack style
    And on each individual font add the crf-grid-stack-1x style
